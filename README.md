# saiapi
SAI Learning model rest API repo


## How to start the app
1. Clone the repo
```
git clone http...
```

2. Install the requirements
```
pip install -r requirements.txt
```

3. Run the api on port 8080
```
py manage.py runserver 8080
```

4. Run the website
```
py manage.py runserver
``` 

## Requirements
`Python 3.12.0`
